# 📚 Library Management System

## 📖 Project Overview

The Library Management System is a console-based application developed using C++. It helps manage library operations such as adding books, searching books, managing members, issuing books, and returning books. The project uses file handling to store data permanently in text files.

---

## 🚀 Features

* Add New Books
* View Available Books
* Search Books by Title
* Add Library Members
* View Member Details
* Issue Books to Members
* Return Issued Books
* View Issued Book Records
* Store Data Using File Handling

## 💾 Data Storage

The project stores data in text files:

* books.txt → Stores book information
* members.txt → Stores member information
* issued.txt → Stores issued book records

## ▶️ How to Run

1. Open the project in Code::Blocks.
2. Build and Run the project (F9).
3. Select options from the menu.
4. Data will be automatically stored in text files.
